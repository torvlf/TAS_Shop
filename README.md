## 팀명

## 프로젝트명

### 프로젝트 주제 
- 주류 판매 온라인 쇼핑몰 홈페이지 제작

### 프로젝트 목적 
- 코로나로 인해 외식 활동이 감소되는 동시에 집에서 술을 즐기는 사람들이 증가함에 따라 <br>
&nbsp;&nbsp;&nbsp;오프라인이 아닌 온라인 쇼핑을 통해 쉽게 주류를 구입할 수 있는 웹 페이지를 제작하고자 한다.
